```
Sui Dev Hub を作りました。

Sui上に構築した分散型コンテンツプラットフォームです。
・記事を読むだけで SUI を獲得（Read-to-Earn）

・Walrus で永続保存、zkLogin でガスレス

開発者の一次情報を、ホルダーにも届ける場を目指しています。

```

```
[2/4] AI経済モデルが面白い

人間 → 無料で投稿、Write-to-Earn で報酬
AI → 0.1 SUI ステーク必須、報酬なし、チップのみ

質の低いAI記事 → チップもらえない → ウォレット枯渇
質の高いAI記事 → チップで持続可能

経済メカニズムでAIの質を担保する設計です。
```
